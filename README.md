

##                                                  AETHER — 3D Landing Page Website
##  INTRODUCTION

        AETHER is a futuristic 3D landing page website built using HTML and CSS with smooth scroll animations and an interactive 3D model.

               This project was created as my first experiment with 3D web design.
               The goal was to learn how modern landing pages combine:

                                                                    > clean UI design
                                                                    > responsive layouts
                                                                    > animated effects
                                                                    > 3D elements

        The website features a dark, futuristic aesthetic with glowing gradients, animated components, and a 3D robot powered by Spline.



## TECHNOLOGIES 
             
             - HTML5
             - CSS3
             - AOS (Animate On Scroll Library)
             - Spline 3D Viewer
             - Responsive Web Design
             - Google/ System Fonts


             
## FEATURES 

              => Futuristic dark UI design
              => Interactive 3D robot model
              => Smooth scroll animations using AOS
              => Gradient glow effects
              => Responsive layout for mobile devices
              => Animated gradient tag box
              => Hover effects on buttons and navigation links
              => Lightweight and fast loading

## THE PROCESS 

              1. Planning the Layout

                              First, I designed the basic structure:

                                                                  > Header
                                                                  > Navigation bar
                                                                  > Main content section
                                                                  > CTA buttons
                                                                  > 3D model placement
              
              2. Creating the UI

                                 I used CSS to:

                                          > build the dark futuristic theme
                                          > create glowing blur effects
                                          > style buttons and typography
                                          > Add hover animations
              
              3. Adding Animations

                              I integrated the AOS library to create:

                                                                  > fade effects
                                                                  > zoom animations
                                                                  > smooth entrance transitions

     This helped the website feel more alive and modern.

              4. Integrating 3D

                          I embedded a Spline 3D scene using:

                               <spline-viewer></spline-viewer>

     This was the most exciting part because it made the landing page feel interactive and professional.

              5. Making It Responsive

                                 Using media queries, I adjusted:

                                                                  > text sizes
                                                                  > spacing
                                                                  > robot scaling
                                                                  > navigation behavior

      The website also works on smaller screens.

## HOW TO RUN THE WEBSITE 
                     
             - Method 1 — Open Directly
                      > Download the project files
                      > Make sure all assets are in the correct folders:
                                                                      * style.css
                                                                      * gradient.png
                      > Open index.html in your browser
              - Method 2 — Using VS Code Live Server
                      > Install VS Code
                      > Install the Live Server extension
                      > Open the project folder
                      > Right-click index.html
                      > Click "Open with Live Server."
